# AI Agent для подбора туров

Архитектура AI агента для платформы бронирования туров.

## Структура

```
ai-agent/
├── prompts/          # Системные промпты
├── tools/            # Инструменты для агента
├── config/           # Конфигурация MCP
└── agent.js          # Основной код агента
```

## MCP серверы

- Context7 MCP - управление контекстом
- Database MCP - работа с данными туров

## Использование

Агент интегрирован с Backend API и Frontend чатом.
